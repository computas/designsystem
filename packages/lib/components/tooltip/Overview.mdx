## How to use

Start by importing the component.

```ts
// Web component
import '@computas/designsystem/tooltip';

// React
import { CxTooltip } from '@computas/designsystem/tooltip/react';
```

To display a tooltip in your app, wrap whatever you want to function as your trigger for the tooltip, as well as the tooltip content,
within the tooltip element.
Then, make sure to assign the "slot" prop of the trigger element to "trigger".
