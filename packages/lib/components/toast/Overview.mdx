## How to get started
Start by importing the component.
``` ts
// Web component
import '@computas/designsystem/toast';

// React
import { CxToast } from '@computas/designsystem/toast/react';
```

Then add it to the root of your app:
``` jsx
<!-- Web component -->
<cx-toast />

<!-- React -->
<CxToast />
```

To display a toast in your app, import and call the `showToast` function whenever you want to display a toast.
The component will automatically display the toast, and hide it after the desired duration.
``` ts
import { showToast } from '@computas/designsystem/toast';

function triggerToast() {
  showToast({
    body: 'This is the alert body',
    duration: 5000
  })
}
```
